## How This Vault Works

1. **Capture** → Everything lands in [[00-Inbox]] first
2. **Process** → Inbox items become fleeting notes → literature notes → atomic/permanent notes
3. **Connect** → Every note links to at least 2 other notes. No orphans.
4. **Compound** → Weekly reviews surface connections, AI synthesis generates insights
5. **Archive** → Completed/inactive items move to [[90-Archives-MOC]]

**Core rule:** One idea per note. If a note has multiple ideas, split it.

---

## Vault Conventions

- **File naming:** `YYYY-MM-DD - Title` for dated notes, `Title` for evergreen
- **Tags:** `#domain/crypto`, `#type/atomic`, `#status/active`, `#project/epm`
- **Links:** `[[wikilinks]]` for internal, `[markdown](url)` for external
- **Frontmatter:** Every note has YAML frontmatter (see templates in [[70-Resources-MOC]])
- **AI Defense Zone:** All raw AI output goes in `60-AI/` — never directly into permanent notes
