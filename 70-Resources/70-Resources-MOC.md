## Plugin Recommendations

### Core (Install First)
- **Templater** — advanced template system with variables and logic
- **Dataview** — query your vault like a database
- **Calendar** — daily note navigation
- **Periodic Notes** — daily, weekly, monthly note templates

### Zettelkasten
- **Breadcrumbs** — hierarchical navigation between notes
- **Strange New Worlds** — see backlinks context
- **Note Refactor** — split notes into atomic pieces

### AI / LLM
- **Smart Connections** — AI-powered note similarity (local or API)
- **Obsidian Copilot** — ChatGPT-like interface inside Obsidian
- **Text Generator** — AI text generation with templates

### Research / Academic
- **Zotero Integration** — import Zotero references
- **Citations** — bibliography management
- **PDF Highlights** — import PDF annotations

### Visualization
- **Excalidraw** — hand-drawn diagrams and sketches
- **Canvas** — Obsidian's native visual whiteboard
- **Graphs View** — built-in, no plugin needed

### Trading / Finance
- **Tracker** — track values over time (portfolio, metrics)
- **Charts**  -- chart rendering from data
