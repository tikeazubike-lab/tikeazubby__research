## Project Template

Use [[TEMPLATE-Project]] for new projects.

Every project note should include:
- **Goal** — one sentence
- **Deliverables** — tangible outputs
- **Deadline** — firm date
- **Status** — Planning / Active / Review / Complete
- **Tasks** — checklist of steps
- **Resources** — links to relevant notes
- **Log** — dated progress entries

## Project Categories

- **Trading** — strategy development, backtesting, live trading
- **Freelance** — client projects, service packages, portfolio
- **Content** — YouTube channels, shorts, affiliate sites
- **PhD** — research projects, paper writing, applications
- **DevOps** — self-hosting setup, automation, infrastructure
- **Learning** — courses, certifications, skill building
