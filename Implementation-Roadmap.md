# Obsidan Zettelkasten — Implementation Roadmap

> Phased setup plan. Follow in order.

---

## Day 1 — Vault Setup

- [ ] Install Obsidian (if not already installed)
- [ ] Open this vault folder in Obsidian
- [ ] Install core plugins: Templater, Dataview, Calendar, Periodic Notes
- [ ] Configure Templater: set template folder to `70-Resources/Templates`
- [ ] Configure Daily Notes: format `YYYY-MM-DD`, folder `50-Journal/Daily`
- [ ] Create your first daily note using [[TEMPLATE-Journal]]
- [ ] Capture 3 things into [[00-Inbox]]

**Time: ~30 minutes**

---

## Week 1 — Foundation

- [ ] Install recommended plugins (see [[70-Resources/Plugin-Setup-Guide]])
- [ ] Process all inbox items from Day 1
- [ ] Create your first 5 atomic notes from existing knowledge
- [ ] Create your first literature note from a recent article
- [ ] Write daily notes every day
- [ ] Do your first weekly review (Sunday)
- [ ] Set up Zotero + Better BibTeX (if doing academic research)

**Time: ~1 hour/day**

---

## Month 1 — Workflow Habituation

- [ ] AI plugins installed and configured (Smart Connections, Copilot)
- [ ] First AI synthesis note in `60-AI/`
- [ ] All domain MOCs populated with initial links
- [ ] First project created in `40-Projects/`
- [ ] Inbox processing is daily habit
- [ ] Weekly review is automatic
- [ ] 20+ atomic notes created
- [ ] 5+ literature notes created

**Time: ~30 min/day (habitual)**

---

## Ongoing — Maintenance

- [ ] Daily: Capture + Process inbox + Daily note
- [ ] Weekly: Review + AI synthesis + MOC updates
- [ ] Monthly: Vault health check + Archive cleanup
- [ ] Quarterly: Review and prune MOCs, update positioning
