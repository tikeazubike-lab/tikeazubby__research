## Processing Checklist

When processing an inbox item:

1. **What is it?** → Idea / Article / Trade / Task / Reference
2. **Does it need action?** → If yes, create task in [[40-Projects-MOC]] or [[50-Journal-MOC]]
3. **Is it worth keeping?** → If yes, create note in appropriate folder
4. **Link it** → Connect to at least 2 existing notes
5. **Tag it** → Add domain + type tags
6. **Delete from inbox** → Move to done or archive
